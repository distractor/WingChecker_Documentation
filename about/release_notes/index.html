    WingChecker Changelog

    Release notes

    - Added feature.
    - Fixed feature.

    v1.0.1.47 13.11.2024

    Release notes

    Setting hidden sheets to very hidden - not visible to users.
    Manual update: reduced text for autotrimmer.

    v1.0.1.46 12.11.2024

    Release notes

    Removed pop-up form for auto-trimmer. Default strategy set.
    Fixed some bugs regarding the auto-trim strategy.

    v1.0.1.43 11.11.2024

    Release notes

    Default values for trim changes and tolerances.
    Trim changes marked with bold, which makes it more clear for the user.
    Auto-trim button added to the worksheet.
    Two auto-trimming strategies added.
    Measuring sheet is zoomed-in by default.
    Enabling cancel button on sheet creation form.

    v1.0.1.42 28.10.2024

    Release notes

    Added default values to trimming options and wing status checks.

    v1.0.1.41 25.10.2024

    Release notes

    Ability to recreate trimmers if required.
    Added trimmer setup when re-evaluated. This enables change of single lines and fixes non-responsive trimmer dropdowns.

    v1.0.1.39 24.10.2024

    Release notes

    Publisher name to installer.
    Tolerances to creation box.
    Help button to current trim options.
    Automated tests.
    Minor changes to quick install script. Removed link start.
    Re-evaluation of already filled Evaluation sheet.
    Creation box pop-up name.
    Disabled maximize/minize windows.
    Split creation into two separated forms.
    Fixed tolerance input validation.
    Line above As shows consequitve attachment id.
    Separate brakes by empty row.
    Added borders in main table to group by main line.
    Remove message after re-evaluation request.
    Measurements and expected lengths from "Measuring" sheet formatted to an integer value.
    Export report to pdf, not directly to printer.
    Smaller create wrokbook button.
    Default measurement set to expected length.
    No exception if evaluation is clicked on blank sheet.
    Actual main line level obtained for generation of Pitch table and calibrated B lines table.

    v1.0.1.30 24.09.2024

    Release notes

    Instructions to install digital certificate.
    Removed quick installer - too dirty.

    v1.0.1.27 23.09.2024

    Release notes

    Quick installer.
    Trim options validator on creation form.
    Creation form position set to center.

    v1.0.1.22 21.09.2024

    Release notes

    Publishing and updates from documentation page.
    Creation box.
    Conditional formatting.
